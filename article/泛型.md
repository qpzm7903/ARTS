# 泛型

为什么用泛型？

定义类型参数，代码重用，编译期检查、类型安全



所以只有保证了类型安全才能往泛型容器里面传。



并且注意，即使是通配符，也要再运行期固定在某个类型上，不可更改。



## 上下界、通配符



以经典例子为例

```java
class Food{
    
}
class Fruit extend Food{
    
}
class Apple extend Fruit{
    
}
class Banana extend Fruit{
    
}
```



List<?>



List<? extend Fruit>

List<? super Fruit>

List<Banana>
