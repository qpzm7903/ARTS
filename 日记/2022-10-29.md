#nginx-upstream #nginx-lb
[upstream](http://nginx.org/en/docs/http/ngx_http_upstream_module.html)

简单负载均衡，默认轮训

```nginx
upstream name{
	server ip:port;
	server ip2:port2;
}

server {
	location / {
		proxy_pass http://name;
	}
}
```


- 保持会话
	- **ip_hash**
		- 不适合ip经常变化的
