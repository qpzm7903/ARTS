## 18 使用GRASP的对象设计示例

CQS原则：命令-查询分离原则
如下例子,将命令和查询的分离

```java
public void roll(){
	this.value = randValue();
}

public int getFaceValue(){
	return this.value;
}

```

但是我们经常这样使用
```java
public int rool(){
	this.value = randValue();
	return this.value;
}

```
